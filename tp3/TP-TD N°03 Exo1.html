1. Formalize the problem as a Markov decision process (MDP). Â¶

The MDP is defined as follows

  • state sapce : The state space consists of all possible agent positions in the 4x4 grid.

    • $S=\{(i,j); i, j â \{0,1, 2,3\} \}$
    • $S= \{(0,0), (0,1), (0,2),(0,3),(1,0),(1,1),(1,2),(1,3),(2,0), (2,1), (2,2), (2,3),(3,0), (3,1), (3,2),(3,3)\}$

  • action space :

    • $A=\{Up, Down, Left, Right\} = \{0,1,2,3\}$
    • $A=\{(\Delta i,\Delta j ) ; \Delta i, \Delta j â\{-1, 0, 1\} and |\Delta i|+|\Delta j|=1 \}$
    • $A=\{(-1,0), (+1,0), (0,-1), (0,+1)\}$

  • Reward function : is function of state and next state

    • R(s, s') = -10, if s=s'
    • R(s,s')=50 if s' is terminate state
    • R(s, s') = -1 otherwise (if sâ s')

  • Transition Probability Function:

    • P(s' | s, a) = 1 if:
      • s=s' and There's a wall in s both in the direction of a and in the opposite direction of a
    • P(s'| s, a) = 0.8 if:
      1. s' is adjacent to s in the direction of action a and There's no wall in s in the direction of a
      2. s=s' and There's a wall in s in the direction of a
    • P(s' | s, a) = 0.2 if:
      1. s' is adjacent to s in the opposite direction of action a and There's no wall in s in the opposite direction of a
      2. s=s' and There's a wall in s in the opposite direction of a
    • P(s' | s, a) = 0 otherwise

  • in summary:

    • If the action would result in hitting a wall, the agent stays in the same state.
    • With probability 0.8, the agent moves in the intended direction if there's no wall in the direction of a.
    • With probability 0.2, the agent moves in the opposite direction if there's no wall in the opposite direction of a.

  • $Î³=0.9$

def transition_probabilities():
    rows,cols=maze.shape[:2]
    nb_action=len(action_space)
    T=np.empty((rows,cols,nb_action), dtype=object)
    
    for i in range (rows):
        for j in range (cols):
            for action in range(nb_action):
                    intended_state=get_nextState((i,j),action)
                    oposite_state=get_nextState((i,j),opposite_action[action])
                    if intended_state== oposite_state:
                        T[i][j][action]={intended_state:1}
                    else:
                        T[i][j][action]={
                             intended_state:0.8, # With 0.8 probability, take the intended action
                             oposite_state:0.2  # With 0.2 probability, take the opposite action
                            }
    return T

def policy_improvement(v):
    #compute q(a|s])
    rows,cols=maze.shape[:2]
    q=np.zeros((rows,cols,len(action_space)))
    for i in range(rows):
        for j in range(cols): 
            for a in range(len(action_space)):
                # for sN in range(state_space):
                #     q[s][a]+=transition[sN][a][s]*(rewards[sN][a][s]+gamma*v[sN])
                q[i][j][a]=np.sum([
                    prob*(reward((i,j),nxtState)+gamma*v[nxtState]) 
                    for  nxtState, prob in T[(i,j)][a].items()
                ])
                

    # get a policy by acting greedily
    policy=np.zeros((rows,cols,len(action_space)))
    for i in range(rows):
        for j in range(cols):
            max = q[(i,j)].max()
            argmax = np.where(q[(i,j)] == max)
            for arg in argmax[0]:
                policy[(i,j)][arg]=1/len(argmax[0])
    return policy

def value_iteration():
    rows,cols=maze.shape[:2]
    value=np.zeros((rows,cols))
    itr=0
    while True:
        valueOld=value.copy() 
        for i in range(rows):
            for j in range (cols):
                value_action=np.zeros(len(action_space))
                for a in range(len(action_space)):
                    value_action[a]= np.sum([
                        prob*(reward((i,j),nxtState)+gamma*valueOld[nxtState])
                        for  nxtState, prob in T[(i,j)][a].items() ])
                        
                value[(i,j)]=value_action.max() 
        if (np.max(np.abs(valueOld-value))<theta):
            print("nb iteration=",itr)
            break
        itr+=1


    policy=np.zeros((rows,cols))
    for i in range(rows):
        for j in range (cols):
            policy[(i,j)]=np.argmax([
                    np.sum([
                       prob*(reward((i,j),nxtState)+gamma*value[nxtState])
                        for  nxtState, prob in T[(i,j)][a].items() ])
                    
                    for a in range(len(action_space))
                ])
    return (policy,value)
