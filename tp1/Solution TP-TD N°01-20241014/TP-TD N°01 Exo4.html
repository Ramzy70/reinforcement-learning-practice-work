Gradient Bandit AlgorithmÂ¶

  • Utilise la preference non pas la valeur de la fonction pour selectionner une action

  • $\large \begin{cases} H_{t+1}(A_t) = H_t(A_t) + \alpha (R_t - \bar{R}_t)(1 - \pi_t(A_t)), \quad \text{and} \\ H_{t+1}(a) = H_t(a) - \alpha(R_t - \bar{R}_t)\pi_t(a), \quad \text{for all } a \neq A_t, \end{cases}$

  • $\large \pi_t(a) \doteq \frac{e^{H_t(a)}}{\sum_{b=1}^k e^{H_t(b)}} $

  • $H_t(a)$ : action preferance

  • $\pi_t(a)$ : action probabilities. Probability of taking action $a$ at time step $t$

  • $\alpha$ : step size

  • $\bar{R}_t$ : average reward (reward baseline) , serve as baseline to compare $R_t$

# Initialize arrays to store results
average_rewards = np.zeros((len(alpha_values), num_steps))
optimal_actions = np.zeros((len(alpha_values), num_steps))

for index, alpha in enumerate(alpha_values):  # for each epsilon value 
    for _ in range(num_experiments):

        #Initialization 
        true_action_values = np.random.normal(0, 1, K) 
        optimal_action=np.argmax(true_action_values)
        
        preferences = np.zeros(K)
        action_probabilities=np.ones(K) / K
        sum_reward = 0

        for step in range(num_steps):
            # print(action_probabilities)
            action = np.random.choice(K, p=action_probabilities) # select an action
            reward = np.random.normal(true_action_values[action], 1) 

            # Update the action preferences.
            sum_reward += reward
            average_reward=  sum_reward/(step+1)
            prAction=preferences[action]
            preferences =preferences - alpha*(reward-average_reward)*(1-action_probabilities)
            preferences[action] = prAction + alpha*(reward-average_reward)*action_probabilities[action]

            # Update action probabilities
            action_probabilities=np.exp(preferences)/ np.sum(np.exp(preferences))
          
            optimal_actions[index, step] += (action == optimal_action) / num_experiments
            average_rewards[index, step] += reward / num_experiments
