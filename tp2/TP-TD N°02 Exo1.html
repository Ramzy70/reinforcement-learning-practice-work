1. Formalize the problem as a Markov decision process (MDP). Â¶

  • Markov Decision Process (MDP) is a mathematical framework for formalizing RL problem by modeling the interaction between an agent and its environment.
  • Formally, A Markov Decision Process is a tuple,<S,A,P,R,Î³> :
    • S: state space
    • A: action space
    • R: provide immediate reward received after taking action a in state s
    • P(sâ²â£s,a): Define the probability of moving from one state s to another s', given a specific action a
    • $Î³ \in [0.1]$ : determines the importance of future rewards compared to immediate rewards

For the 3x3 grid world, the MDP is defined as follows

  • state sapce : The state space consists of all possible agent positions in the 3x3 grid.

    • $S=\{(i,j); i, j â \{1, 2,3\} \}$
    • $S= \{(1,1), (1,2), (1,3), (2,1), (2,2), (2,3), (3,1), (3,2), (3,3)\}$

  • action space :

    • $A=\{Up, Down, Left, Right\} = \{0,1,2,3\}$
    • $A=\{(\Delta i,\Delta j ) ; \Delta i, \Delta j â\{-1, 0, 1\} and |\Delta i|+|\Delta j|=1 \}$
    • $A=\{(-1,0), (+1,0), (0,-1), (0,+1)\}$

  • Reward function : is function of state and action

    • R(s, a) = -1 if the action a is non-permitted from state s
    • R(s, a) = r, where r is the value given in the reward table

  • Transition Probability Function:

    • P(s' | s, a) = 1 if s' is adjacent to s in the direction of action a
    • P(s' | s, a) = 1 if a is non-permitted action and s' = s
    • P(s' | s, a) = 0 otherwise

  • $Î³=0.7$

def get_returns(returns,trajectory):
    G = 0
    for t, sar in enumerate(reversed(trajectory)):
        s, a, r = sar
        G= r + gamma * G
        if s in returns:
            returns[s].append(G)  
        else:
            returns[s] = [G] 
    return returns

def monte_carlo_policy_evaluation(policy,nb_episodes):
    #initialization of the value function 
    value_function=np.zeros(shapeValue)
    returns={}
    for i in range(nb_episodes):
        trajectory= get_trajectory(policy)
        returns = get_returns(returns, trajectory)
    
    for i in range(nb_rows):
        for j in range (nb_cols):
            if (i,j) in returns:
                value_function[(i,j)] = np.round(np.mean(returns[(i,j)]), 2)  # rounds the  mean two decimal places
    
    return value_function
